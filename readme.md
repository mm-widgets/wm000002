# 倒计时

## 属性

### until

倒计时的截止时间

### style

组件样式

### size

组件大小

### timeLabelStyle

时间标签样式

### separatorStyle

分隔符样式

### digitStyle

数字样式

### digitTxtStyle

数字文本样式

### timeToShow

显示的格式[‘D’,’H’,’M’,’S’]

### timeLabels

文本要在时间标签中显示(如：{ d: ‘天’;h: ‘时’;m: ‘分’;s: ‘秒’; })

### showSeparator

是否显示分隔符

## 事件

### onFinish

倒计时结束事件

### onChange

时间改变，注:如果界面到后台之后，事件不触发
